FEATURES

What it does

01

Mud Clean

The core feature. Strips low-mid buildup from the reverb tail in real time — dial it up and the muddiness disappears while the space stays intact. Run longer decays without clouding the mix.

02

Pitch Clean Mode

Keeps the reverb tail harmonically aware of the input pitch. The tail tracks what you're playing rather than smearing across random frequencies — more musical, less wash.

03

EQ & High Damp Shaping

A built-in EQ display lets you shape the frequency content of the tail directly. High Damp rolls off the top end as the reverb decays for a natural room feel.

04

Gate & Duck Input

Gate the tail so it closes cleanly instead of washing into the next phrase. Duck Input automatically pulls the reverb back when the dry signal hits, keeping the source upfront.

05

LFO Modulation

Mod Depth and Mod Rate add slow movement to the reverb tail — subtle chorus-like drift that keeps long reverbs from sounding static. Damp Speed controls how fast the modulation affects the tail.

06

Chamber Types & Presets

Multiple chamber types change the character of the space. Presets like Clean Bloom and Rainy Day give fast starting points — copy and save your own to build a library.
