Say So

VOICE CONTROL / ABLETON LIVE

Talk to your DAW. Say So puts full Ableton Live control on your voice — transport, tempo, tracks, clips, mixing, and hands-free chord composing. Type a correction when it mishears and it learns what you meant. Say "over" to end a command. Say "pause" when you're singing so nothing fires by accident.

FEATURES

What it does

01

Constrained Grammar + Learning

The recognizer only hears words it understands — plus your track names. When it mishears, type what you actually said and Say So learns the correction, improving accuracy on your voice over time.

02

Full Ableton Control

Transport, tempo, metronome, loop, undo, scenes, clips, mute, solo, arm, volume, pan, plugin loading — all on your voice. Works via AbletonOSC running as a Control Surface in Live.

03

Hands-Free Composing

Say a chord progression and Say So writes the M2TM Notes syntax, creates a clip, and types it in — all without touching the keyboard. "Make a MIDI clip on drums scene one, C major G major A minor each one bar."

04

Smart Track Renaming

Say "rename drums" — Say So switches to open vocabulary, asks for the new name, you speak anything, it renames and snaps the constrained grammar back on. One-shot "rename X to Y" still works for common words.

05

Plugin Loading

Say "put Operator on drums" or "add reverb to bass" — Say So selects the track and navigates Live's browser to load the plugin in one sentence. Live must be the active window for browser automation to work.

06

Live Track Sync

Track names are added to the vocabulary automatically whenever they change. Say "seven audio" to match Live's default naming. Say "refresh tracks" to force a re-read if names get out of sync.

07

Mic Gate & "Over" Mode

Say "pause" before you sing and Say So stops listening — nothing fires while you perform. Say "resume" to bring it back. Toggle the "wait for over" setting and Say So won't run a command until you say "over", giving you time to finish a longer phrase without accidental triggers.

08

Type to Teach

When Say So mishears a command, just type what you actually meant to say. It logs the correction, maps your voice to the right command, and gets more accurate on your accent and speech patterns over time. The more you correct it, the less you have to.

09

AbletonOSC + SaySoBridge

Under the hood, Say So uses AbletonOSC to send OSC messages directly into Ableton Live — controlling transport, mixer, clips, and more without MIDI or keyboard shortcuts. SaySoBridge sits alongside it as a second Control Surface, handling plugin loading directly through Live's API so nothing has to touch the search bar.

SETUP

Getting started

01

Install AbletonOSC

Download AbletonOSC and drop it into Documents\Ableton\User Library\Remote Scripts\. In Live's preferences, select it as a Control Surface and restart Live.

02

Install SaySoBridge

SaySoBridge handles plugin loading without requiring you to type into Ableton's search bar. Install it as a Control Surface alongside AbletonOSC — drop it into Documents\Ableton\User Library\Remote Scripts\ and select it in Live's preferences.

03

Install dependencies

Open a terminal in the Say So folder and run pip install -r requirements.txt. Requires Python 3.8+.

04

Run Say So

Double-click run.bat — or run python sayso.py from the terminal. The status dot turns green when your mic is active and Ableton is connected.

05

Calibrate M2TM (optional)

To use hands-free chord composing, open a clip with the M2TM Notes text box visible, hover your mouse over it, and say calibrate. Remembered across sessions.

NOTE — PLUGIN LOADING

With SaySoBridge installed, commands like "put Operator on drums" load plugins directly without touching the search bar. If Live runs as administrator, run run.bat as administrator too.

NOTE — MISHEARING

Everything heard is logged — look for heard: "..." when no command matched. Consistent mishearing? Add an alias to _ALIASES in the script, or check that the word is in _GRAMMAR_BASE.

COMMANDS

Voice reference

TRANSPORT & TEMPO

"play" / "stop" / "continue"Transport controls
"record on" / "record off"Session record toggle
"tempo one twenty eight" / "tempo up" / "tempo down"Set or nudge BPM
"metronome on/off" / "loop on/off"Metronome and loop toggle
"undo" / "redo"Edit history
"pause" / "resume"Mic gate — stop/start listening (use when singing)
"over"End of command (when "wait for over" is toggled on)

SCENES & CLIPS

"launch scene three" / "next scene" / "previous scene"Scene navigation and launch
"fire clip two on drums"Launch a clip by index on a named track
"stop all clips"Stop all running clips

MIXER

"mute drums" / "unmute bass"Mute / unmute by track name or "track three"
"solo drums" / "unsolo drums"Solo toggle
"arm drums" / "disarm drums"Record arm toggle
"volume up on bass" / "set volume fifty on drums"Fader control (85 = 0 dB)
"pan left on keys" / "pan center on drums"Panning
"select drums"Select track

TRACKS & PLUGINS

"new audio track" / "new midi track"Create track
"rename drums" → say any nameTwo-step rename, open vocabulary
"rename drums to synth bass"One-shot rename for common words
"delete track drums"Delete track (undo restores)
"put operator on drums" / "add reverb to bass"Select track + load plugin from browser
"add plugin operator"Load plugin on currently selected track
"refresh tracks"Re-read track names into vocabulary

M2TM CHORD COMPOSING

"make a midi clip on drums scene one, C major G major A minor each one bar"Creates clip, types chord syntax, hits Enter
"compose C major G major each one bar"Builds syntax from chord phrase
"a C major chord for one bar then D minor for two bars"Per-chord durations
"notes C D E G each one beat" / "melody C octave four D E"Single-note lines
"... then a rest for two beats"Rests
"C major loud, G major softly"Velocity control
"smooth ..." / "like a pianist"Voice leading (AV LE)
"arpeggiate ..." / "strummed ..."Arpeggio and strum modes
"calibrate"Remember M2TM text box position (hover mouse first)
"clear text"Wipe the M2TM box
